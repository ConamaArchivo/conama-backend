<h2>To run it locally:</h2>

1.  Clone this repo
```bash
git clone git@github.com:ConamaArchivo/server.git
```

2. Change directory
```bash
cd server
```

3. Install dependencies
```bash
npm install
```

3. Run as development
```bash
npm run dev
```
